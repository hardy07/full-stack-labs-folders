Node.js Installation on Windows and HTML Basics

Installing Node.js on Windows 10

Step 1: Download Node.js

Visit the official Node.js website: https://nodejs.org/en/download/

Step 2: Running the Node.js Installer

  1. Double-click on the .msi installer.
  2. The Node.js Setup Wizard will open.
  3. Click "Next" on the welcome screen.
  4. Accept the End-User License Agreement and click "Next."
  5. Select the Destination Folder and click "Next."
  6. Click "Next" on the Custom Setup screen.
  7. Click "Install" to proceed.
  8. Do not close the installer until the installation is complete. Click "Finish" once done.

Step 3: Verify Installation

Open Command Prompt or PowerShell and run the following command:

C:\Users\Admin> node -v

If installed correctly, this will display the Node.js version.

Step 4: Updating npm

Run the following command to update npm:

npm install npm --global

Step 5: Running a Sample Application

Save the following code as server.js and run node server.js in your terminal:


const { createServer } = require('node:http');
const hostname = '127.0.0.1';
const port = 3000;

const server = createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello World');
});

server.listen(port, hostname, () => {
  console.log(`Server running at http://${hostname}:${port}/`);
});

Open a browser and go to http://127.0.0.1:3000 to see "Hello World".

Step 6: Running JavaScript Code in VS Code

Create a folder, then create a file named First.js with the following content:


const x = "My first web page";
console.log(x);

Run the code in the terminal with:

node .\First.js
